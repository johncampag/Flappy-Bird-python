# Flappy Bird — Python

A fully functional Flappy Bird clone built in Python using the CMU Graphics library.

## Features
- Physics-based bird movement with velocity and rotation
- Randomized pipe generation with consistent gap sizing
- Score tracking with persistent best score
- Pause and resume functionality
- 3 selectable themes that reskin the bird, pipes, backgrounds, and all UI elements
- Animated game over sequence with sliding UI elements

## Built With
- Python
- CMU Graphics Library

## How to Play
- Click or press Space to flap
- Avoid the pipes
- Use the pause button to pause
- Click ??? on the game over screen to switch themes
